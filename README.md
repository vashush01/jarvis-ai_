# 🤖 Jarvis AI - Voice Controlled Desktop Assistant

Jarvis AI is a Python-based intelligent voice assistant that automates daily desktop tasks using voice commands. It can answer questions, open applications, search the web, play music, and interact with AI models to provide intelligent responses.

---

## 🚀 Features

- 🎤 Voice Command Recognition
- 🗣️ Text-to-Speech Response
- 🌐 Web Search
- 🔍 Google Search Integration
- ▶️ YouTube Search
- 📂 Open Desktop Applications
- 📝 Clipboard Operations
- 🤖 AI-Powered Conversations
- ⏰ Date & Time Information
- 📧 Basic Task Automation
- 💻 Easy-to-Use Command Line Interface

---

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/vashush01/jarvis-ai.git
```

### Move into Project

```bash
cd jarvis-ai
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Environment

Windows

```bash
.venv\Scripts\activate
```

Linux / Mac

```bash
source .venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python main.py
```
